Singular or Non-singular Matrix

To find a matrix is whether Singular or Non-Singular first we need to find it's Determinant value.
If the value of determinant is Zero then it is a Singular Matrix.
If the value of determinant is Non-Zero then it is a Non-Singular Matrix.
